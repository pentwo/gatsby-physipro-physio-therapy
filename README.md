# PhysiPro Physiotherapy

A bilingual (English/Chinese) website for PhysiPro Physiotherapy clinic built with Gatsby 5.

## Tech Stack

- **Framework**: Gatsby 5 with TypeScript
- **UI**: Mantine v7
- **Styling**: CSS with custom properties
- **i18n**: React Context with localStorage persistence

## Features

- Full English/Chinese (中文) localization
- 6 specialty services
- 8 treatment modalities
- Contact page with clinic locations (Duncraig & Northbridge)
- Responsive design

## Development

```bash
npm install
npm run develop
```

## Build

```bash
npm run build
npm run serve
```
